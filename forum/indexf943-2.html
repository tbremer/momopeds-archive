I would go down the whole throttle cable from top to bottom and make sure it's not sticking on anything.  Run some lube down in the cable itself to make sure it's not sticking on anything.  Is this a new cable?  If it's old it might be starting to fray somewhere and if one of those little wires is sticking out inside the cable it can catch and drag on the ridges of the cable sheath.  If that makes any sense.

I almost guarantee you can find any QT50 bolts you need around the shop.
