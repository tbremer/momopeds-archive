Things I'm aware of:
1) It used to look like crap when accessed from a phone, I think because I made the box a fixed size rather than a relative size. That pushed the images and a lot of the text outside the intended content area. I googled for a while and found the "overflow" property that is supposed to make it scroll. If I leave this set to "scroll," it works on my phone, but then the scroll bars are visible on my computer, which I don't like (especially since it doesn't need to scroll on my computer screen). I changed it around a few times, and it's on "auto" now. This seems to make it display properly on my phone and it eliminates the unnecessary visible scroll bars on my computer, but I've only tested it on my laptop (Chrome) and my phone (Safari). I could quit being lazy and go test it on other computers, or I can just ask you guys.

2) I'm adding in file size info to each link, and I'm only through the C's right now. The rest will get there eventually.

3) Some of the files are rather large. I'm probably going to chop those down and offer them as multiple parts for anyone who's on a slower connection. Unless anyone has a better idea about making it more accessible to people with slow internetz?

4) Yes, I did steal design ideas from MopedStL. That's called, "It was clean and simple and it worked, and I didn't know enough to come up with my own ideas when I started. By the time I learned enough, I didn't want to change it." Imitation is the highest form of flattery, right?

What are you using to make the PDFs?  Acrobat lets you choose different formats for the photos and different quality levels.

TIFF is really just a container format.  One advantage it has over PNG is that you can do multiple pages within one file, but that's not that big of a deal.  TIFF is more flexible than PNG because you can do different compressions within the TIFF.  They are both lossless though so it doesn't matter which one you use for editing.  I'd stick with PNG.

Side note:  PNG is better for line drawings or images where the difference between one pixle and another is going to be sharp.  JPEG is for photos, but only if you are never going to edit them again.  Each time a JPEG is changed and saved the quality gets worse.
