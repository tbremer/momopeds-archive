stop.   harry, the lever isn't a kick start, it's a wind up, like a wind up toy,,,,get it?

look above the foot lever. there is a lever that looks like a brake cable goes to it. pry it forward with a screw driver, and it'll release the wind up foot lever and make the engine spin.

this lever is on the white side, it's sticking up a little bit out of the transmission, between where the carb is and the rear fender. 
       pry it forward.

if and when you get a starter cable/rear brake cable,, it hooks up there.

wait a minute.... you have the cable, and it's on there?  in that case just pull the rear brake lever on the handlebars. that wind up shit breaks, so go easy on it. listen, don't open the transmission unless you know for sure that you broke it. if it's broke, then see if one of them other engines has a good one and transfer everything over to the good one.

the start lever takes 3 pumps to wind it up and it should return a little each time.

that cable has to be adjusted forward enough to pop start the engine, but not so tight that the brakes are locked up. you'll have one hand adjusting the cable tension and the other pulling the handle bar lever.
